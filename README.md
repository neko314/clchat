# Clchat

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'clchat'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install clchat

## Usage

```shell
$irb
>> require 'clchat'
=> true

>> Clchat.answer("happy")
```
